Riddle Game (Java Console Project)

A simple console-based Riddle Game built using Java.
The player is presented with random riddles, gets hints for wrong answers, and earns points for every correct response.
Perfect for beginners, Java learners, and college mini-projects.

Features

Random riddles every round
Hint option when the user answers incorrectly
Three riddles stored in arrays
Score tracking
Play again option
Beginner-friendly Java code (uses loops, arrays, conditions, Scanner class)

Technologies Used

Java (JDK 8 or above)
Scanner class for input
Basic control structures (loops, if-else)

Concepts Covered (Good for Students)

Arrays
Random number generation (Math.random())
Loops (while)
Conditional statements
Taking input using Scanner
String comparison in Java



Project Outcome

This project helps students understand:
How to build interactive console applications
Logic building with arrays and loops
How to handle user input
How to generate random questions
